Протокол Шамира.

Запуск из cmd:
run.bat

При каждом запуске программы генерируется случайный ключ, случайный модуль.
В итоге Bob получает исходный секретный ключ Alice, который третья сторона узнать за разумное время не сможет.


Вывод программы:

Секретный ключ Alice K=1948
Alice выбирает случайное простое P=47017 и отправляет его Bob
Alice выбирает 0<a<p-1 такое, что (a, p-1)=1. a=5
Alice передает K^a mod p = 33503
Bob передает (K^a)^b mod p = 1748
Alice передает ((K^a)^b)^a^(-1) mod p = 33503
Bob получает ключ (((K^a)^b)^a^(-1))^b^(-1) mod p = 1948
