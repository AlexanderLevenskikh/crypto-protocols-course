Схема разделения секретов.

Реализована схема разделения секретов Шамира.
Значения (n, k) заданы в исходном коде программы, секрет - 42 (например)
Модуль выбирается случайно, далее случайным образом генерируются коэффициенты одночленов с ненулевыми степенями
Многочлен имеет максимальную степень k-1
Так же, случайным образом выбираются значения x точек и точки (x, y) отправляются абонентам.
Если людей для получения правильного секрета недостаточно (< K), тогда секрет получится некорректный
иначе - получается корректный результат.

Вывод программы:

Сгенерирован многочлен: 42 + 4x^1 + 4x^2 + 1x^3
Сгенерированы фрагменты секрета: (20, 9722), (42, 81354), (29, 27911), (59, 219581), (90, 761802), (74, 427466)
Результат группы из 1, 2 и 4 человека (недостаточно людей для получения секрета): 19483
Результат группы из 2, 3 и 4, 6 человека (достаточно людей для получения секрета): 42
Результат группы из 1, 2, 3, 4 и 5 человека (людей для получения секрета с избытком): 42
