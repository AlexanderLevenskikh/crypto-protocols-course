Головоломка Меркла.

Запуск из cmd:
run.bat

При каждом запуске генерируется 10 сообщений вида
key: число, id: число, где key - ключ для шифрования алгоритмом Blowfish, id - идентификатор сообщения. Все числа случайные
Все эти сообщения шифруются при помощи алгоритма Blowfish слабым ключом (значения от 1 до 100)
Alice берет случайное сообщение и расшифровывает его (брутфорс ключей)
В итоге, у Alice есть ключ и id сообщения, далее она может отправить этот id Бобу и зашфированное этим ключом секретное сообщение
Боб по id легко определит, какой ключ. Злоумышленнику же придется перебрать все сообщения.
В реальных условиях сообщений может быть очень много, и ключи - большего размера.
