Реализация нейросетевого протокола для n = 8

На каждом шаге Alice и Bob синхронно тренируют свои нейронные сети.
Пока не произойдет синхронизация (с большой вероятностью выходы НС совпадают - не менее 100 попыток подряд),
НС будут тренироваться. В конце Alice и Bob имеют обший ключ.

Вывод программы:

Alice и Bob начинают тренировать нейронные сети
После первой итерации
Входная строка Alice: 00101001
Выходная строка нейронной сети Alice: 00010001
Входная строка Bob: 11100001
Выходная строка нейронной сети Bob: 00010001
Конец итерации

После 100 итерации
Входная строка Alice: 10111001
Выходная строка нейронной сети Alice: 00010001
Входная строка Bob: 01000010
Выходная строка нейронной сети Bob: 00010010
Конец итерации

Синхронизировались на 6082 итерации
Входная строка Alice: 11101110
Выходная строка нейронной сети Alice: 00010110
Входная строка Bob: 11111000
Выходная строка нейронной сети Bob: 00010110


Alice сгенерировала ключ: 00010110
Bob сгенерировал ключ: 00010110
