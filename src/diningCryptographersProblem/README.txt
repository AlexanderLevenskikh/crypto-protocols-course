Завтрак трех криптоаналитиков.

Запуск из cmd:
run.bat

Программа демонстрирует решение задачи обедающих криптографов.
Алгоритм:
Каждый криптограф подбрасывает монетку со своим соседом в тайне от третьего. Получается 3 общих секрета.
Секрет Alice и Bob
Секрет Alice и Carol
Секрет Carol и Bob
Далее, если криптограф не платил за завтрак, то он вычисляет XOR своих секретов, иначе - обратную величину (логическое равенство)
Получается, если никто не платил - значение будет равно 2(Xab XOR Xac XOR Xbc) === 0
В любом другом случае - будет равно 1

Вывод программы:

Запустим с маской 010.
Входные данные: заплатил Bob
Подбрасывание монет...
Общий секрет Alice и Bob: true
Общий секрет Alice и Carol: true
Общий секрет Carol и Bob: true
XOR Alice: false
XOR Bob: true
XOR Carol: false
Результат: true. Заплатил кто-то из криптографов


Запустим с маской 000.
Входные данные: заплатил АНБ
Подбрасывание монет...
Общий секрет Alice и Bob: true
Общий секрет Alice и Carol: false
Общий секрет Carol и Bob: true
XOR Alice: true
XOR Bob: false
XOR Carol: true
Результат: false. Заплатило АНБ
